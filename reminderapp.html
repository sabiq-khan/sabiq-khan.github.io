Reminder App

The Challenge: I had to build a desktop application in Java that allowed users to schedule tasks they wanted to complete at a certain time. The app had to allow users to enter details about these tasks such as a deadline, a description, and when they wanted to recieve a notification reminding them about their task.

Tools Used:
                 Front-end: JavaFX
                 Back-end: Java

Development and Role: I took an MVC approach to building this application, with a different Java class for handling each. My Model class created an ObservableList of Task objects, each of which stored information about a task such as deadline, description, priority, and reminder time. My View class used JavaFX to create a interactive GUI that presented users with text fields where they could enter information and sent them notifications about their tasks whenever appropriate. My Controller class allowed the Model and View to communicate; information entered into the GUI would result in updates to the Model and back-end logic in the Model could result in changes to the GUI.
