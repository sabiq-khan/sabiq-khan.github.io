Java Compiler

The Challenge: I had to build a compiler in Java that itself could compile Java source code.

Tools Used:
                 Languages: Java

Development and Role: I approached this problem through recursive-descent parsing. First, I created a Lexer class that would break source code down into its basic syntactical elements. I implemented recursive-descent parsing by creating a Parser class that organized these syntactical elements into a binary tree. A preorder traversal of this tree produced intermediate three-address code that could then be translated to assembly and executed.
