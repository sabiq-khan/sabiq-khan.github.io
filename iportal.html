Intranet Portal

The Challenge: While I was an employee at an IT solutions company, we had a client who wanted us to build an intranet portal web application for them. The client was a major hardware manufacturer who wanted to use this application to track employee assignments, projects, and manufacturing workflows.

Tools Used:
                 Front-end: HTML, SCSS, Bootstrap, TypeScript, Angular
                 Back-end: C#, ASP.NET, MySQL

Development and Role: I worked on both the front and back end of this application. On the front-end, I used HTML, SCSS, and Angular to build interactive web pages. I created forms into which users could enter input and be shown the status of ongoing projects and manufacturing processes. On the back-end, I handled POST and GET requests using C# in ASP.NET and wrote queries to the database using LINQ. Additionally, I helped style the UI, layout, and overall appearance of the front-end using SCSS and Bootstrap and created icons and logos myself wherever necessary.
